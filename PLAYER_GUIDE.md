# 📜 Transportation 冒险者坐骑指南

欢迎来到 **Transportation** 的世界！这是一个致力于让每一位冒险者都能拥有专属、忠诚且便携的坐骑系统的指南。在这里，你的爱马不再是随处可见的牲畜，而是与你缔结契约的灵魂伙伴。

---

## 🌟 核心概念 (Core Concepts)

在开始之前，请熟悉以下三个概念，它们将贯穿你的冒险旅程：

1.  **契约 (Contract)**: 你与生物之间建立的灵魂连接。只有缔结契约后，它才真正属于你，并受到系统的保护。
2.  **兽栏 (Stable)**: 一个神奇的亚空间（数据库），当你的坐骑不在此处时，它们都在兽栏中休息。无论你在世界的哪个角落，都能从兽栏召唤它们。
3.  **信物 (Token/Key)**: 一个注入了魔力的物理媒介（如马铠、命名牌或木棍）。持有信物，你就可以随时随地召唤或收回对应的坐骑。

---

## 🐴 第一章：缔结契约 (Binding)

### 1. 驯服伙伴
首先，你需要按照原版游戏的方式驯服一个生物（例如驯服一匹马、驴或羊驼）。

### 2. 签订契约
将准星对准你驯服的生物，咏唱以下咒语（输入指令）：

*   **基础契约**:
    ```
    /tra bind
    ```
    *系统会自动将其注册为你的坐骑。*

*   **命名契约** (推荐):
    如果你有多匹马，为了区分它们，建议给它们起个代号：
    ```
    /tra bind HORSE:赤兔
    ```
    *这样你就有了一匹代号为“赤兔”的马。*

> **✅ 成功后**: 坐骑会被纳入你的**兽栏**，并且它的所有属性（速度、跳跃力、生命值、装备、甚至身上的药水效果）都会被永久记录。

---

## 🗝️ 第二章：制作信物 (Keys)

虽然你可以用指令召唤坐骑，但拥有一个物理“信物”会更加方便和帅气。

### 1. 制作方法
1.  **准备物品**: 手里拿一个你喜欢的物品（推荐：马铠、鞍、命名牌，或者木棍）。
2.  **注入魔力**: 输入指令：
    ```
    /tra keybind <坐骑代号>
    ```
    *例如：`/tra keybind HORSE` 或 `/tra keybind 赤兔`*

3.  **完成**: 手中的物品现在变成了**信物**！上面会显示坐骑的详细信息。

### 2. 使用信物
*   **召唤 (Summon)**: 手持信物，**右键点击** 空气或地面。
    *   *效果：坐骑瞬间出现在你面前。*
*   **收回 (Recall)**:
    *   **近距离**: 手持信物，**右键点击** 坐骑。
    *   **远距离**: 手持信物，**Shift + 右键点击** 空气或地面。
    *   *效果：坐骑化作光芒回到兽栏休息。*
*   **骑乘**: **空手**（或手持非信物物品）右键点击坐骑。

> **⚠️ 注意**: 不要手持信物去骑马，那会触发“收回”操作！

## ⚔️ 第三章：日常管理 (Management)

### 1. 查看兽栏
想知道你有哪些坐骑？它们状态如何？
```
/tra list
```
*你将看到所有坐骑的状态：[出战中]、[休息中] 或 [重伤]。*

### 2. 远程召唤/收回 (无信物)
如果你没带信物，或者信物丢了，可以直接使用咒语：
*   **召唤**: `/tra out <代号>`
*   **收回**: `/tra in <代号>`

### 3. 救治坐骑 (Revive)
如果你的坐骑不幸战死，不要悲伤，契约的力量可以复活它！
1.  查看 `/tra list`，确认状态为 **[重伤]** (Destroyed)。
2.  输入指令支付医疗费复活它：
    ```
    /tra fix <代号>
    ```
    *例如：`/tra fix 赤兔`*

---

## 🤝 第四章：社交与交易 (Social)

### 1. 坐骑转让
你可以将精心培育的宝马赠送或交易给其他玩家：
```
/tra transfer <代号> <玩家名>
```
*注意：转让后，你手中的信物将失效，对方成为新的主人。*

### 2. 解除契约
如果你想放生它，或者腾出兽栏空间：
```
/tra unbind <代号>
```
*警告：此操作不可逆，坐骑将不再属于你。*

---

## 🛡️ 安全与保护 (Protection)

*   **防盗**: 只有你可以骑乘你的坐骑（除非你将其转让）。
*   **防篡改**: 其他人无法使用命名牌更改你坐骑的名字。
*   **属性保留**: 
    *   无论是收回、召唤还是转让，坐骑的 **速度**、**跳跃力**、**装备**（马铠/箱子）、**驯服状态** 以及 **药水效果** 都会完美保留。
    *   *小技巧：你可以给马喝下迅捷药水然后收回，下次召唤时它依然拥有加速效果！*

## ❓ 常见问题 (FAQ)

**Q: 我的信物（物品）丢了怎么办？**
A: 没关系！信物只是一个媒介。你的坐骑安全地保存在数据库中。你可以拿一个新的物品，使用 `/tra keybind` 重新绑定即可。

**Q: 为什么提示“兽栏已满”？**
A: 每个玩家可拥有的坐骑数量是有限的（由管理员设置）。你需要使用 `/tra unbind` 释放一些不需要的坐骑。

**Q: 我可以用命名牌改名吗？**
A: 可以，作为主人，你可以自由使用命名牌。但其他玩家无法修改你坐骑的名字。

**Q: 跨世界可以召唤吗？**
A: 当然！兽栏连接着所有世界。你可以在主世界收回它，然后在地狱将其召唤出来。

## 📜 咒语速查表 (Commands)

| 咒语 (Command) | 参数 | 说明 |
| :--- | :--- | :--- |
| `/tra bind` | `[model:id]` | 将准星对准的生物缔结为坐骑 |
| `/tra keybind` | `[model]` | 将手中物品制作为信物 |
| `/tra list` | 无 | 查看所有坐骑及其状态 |
| `/tra out` | `<model>` | 从兽栏召唤坐骑 |
| `/tra in` | `<model>` | 将坐骑收回兽栏 |
| `/tra fix` | `<model>` | 复活已死亡的坐骑 |
| `/tra transfer` | `<model> <player>` | 将坐骑转让给他人 |
| `/tra unbind` | `<model>` | 解除契约（删除坐骑） |
| `/tra unkeybind` | 无 | 清除手中物品的信物属性 |
